# AI Usage

Track your AI subscription and API usage limits in one place: Claude, ChatGPT, and Gemini, plus their developer APIs. Connect several providers, and several accounts under each one, and see how much of your session limits and quotas you have left and when they reset.

## Download

Get the latest APK from the [Releases page](https://github.com/Neoo-Blue/ai_usage/releases/latest). To install, enable "install unknown apps" for your browser or file manager, then open the downloaded file.

## What it does

- **Multiple providers and multiple accounts.** Connect a personal and a work ChatGPT, a Claude Pro account, a Gemini account, all at once. Two accounts on the same provider never overwrite each other.
- **Two kinds of account:**
  - **Subscription:** sign in through a secure in app browser. The app keeps only your session, on your device.
  - **Developer API key:** paste a key and see live remaining tokens and requests straight from the provider.
- **Everything stays on your device.** Your logins live in the system keychain, never in a database and never uploaded. The app talks only to each provider directly.
- **Background sync** keeps your usage and reset times current.

## Home screen widget (coming)

A customizable home screen widget with six themes: Minimalist, Elegant, Futuristic, Neumorphic, Retro, and Adaptive, which follows your phone's own colors. You choose the account, the metrics, and the theme for each widget you place.

## Status

Early build. Working today: connecting accounts, developer API key live numbers, and subscription sync. The native home screen widget is the next piece.

## Privacy

Your provider logins never leave the device. Nothing is sent to any server the app runs, because there is no such server. The app is a direct client to Claude, ChatGPT, and Gemini.
